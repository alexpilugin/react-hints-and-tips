Create simple Express.js app with React First step is install Node.js globally: https://nodejs.org Download the package and install it from the .pkg by following along the install wizard which will install both node and npm, npm is Node Package Manager which allows for installs of additional packages for node.js. Create folder for your application, using following commands in Terminal: $ mkdir myappNote: most npm commands should be ran in your project directory. Go inside the folder:$ cd myapp Use the npm init command to create a package.json file for the application. $ npm init
Installing packages with npm: Installing expressjs:http://expressjs.com Install Express in the application's folder with saving it in the dependencies list: $ npm install express --save Express application generator: It's possible to use Express application generator:$ npm install express-generator -g Install React with npm: $ npm install --save react react-dom Install Browserify globally: $ npm install -g browserify The -g flag makes npm install the package as global, meaning you get access to the browserify command from the command line. You may need to use sudo to run this. https://github.com/babel/babelify Install the babelify package, which allows browserify to use babel to compile JSX code: $ npm install --save babelify babel-preset-react babel-preset-es2015 Edit package.json file (two scripts separated by '|'): "scripts": { "start": "browserify src/app.js -o public/app.js -t [ babelify --presets [ es2015 react ] ] | node index.js", "test": "echo \"Error: no test specified\" && exit 1" }, It's possible watching file changes with nodemon:npm install --save-dev nodemon And in package.json: "scripts": { "start": "nodemon index.js", } Create index.js file (for nodejs): $ echo /*index.js*/ > index.js Create public subfolder: $ mkdir public Create source subfolder: $ mkdir srcAnd create app.js file in src folder Create empty html file inside public folder: $ echo > public/index.html Create empty app.js file inside src folder (will be transformed into public folder): $ echo > src/app.js
Steps: 1. Create project folder and move in 2. $ npm init 3. $ npm install express --save 4. $ npm install --save react react-dom 5. $ npm install --save babelify babel-preset-react babel-preset-es2015 6. Correct package.json file (add script): "start": "browserify src/app.js -o public/app.js -t [ babelify --presets [ es2015 react ] ] | node index.js", 7. Create index.js file (for nodejs) 8. Create public folder with index.html file 9. Create src folder with app.js file 10. Launch app: $ npm start
